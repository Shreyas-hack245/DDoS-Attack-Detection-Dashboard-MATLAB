# DDoS Attack Detection Dashboard

## Description

This project is a MATLAB-based simulation for detecting DDoS (Distributed Denial of Service) attacks. It monitors network traffic in real time and identifies abnormal traffic spikes that may indicate an attack.

## Features

* Real-time traffic monitoring
* DDoS attack detection
* Attack severity classification
* Network health monitoring
* Source IP simulation
* Traffic distribution analysis
* Detection summary report

## Technologies Used

* MATLAB

## How It Works

1. Simulates network traffic.
2. Monitors packet rates continuously.
3. Detects traffic exceeding the threshold.
4. Classifies traffic as Normal or Attack.
5. Displays results using graphs and alerts.

## Output

* Live traffic graph
* Attack detection alerts
* Pie chart of traffic distribution
* Bar chart summary

## Conclusion

This project demonstrates a simple and effective approach to detecting DDoS attacks using MATLAB simulation and visualization techniques.
